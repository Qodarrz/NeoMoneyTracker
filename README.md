# NeoWestMoney

NeoWestMoney is a monorepo containing the Backend (Go), Frontend (Next.js), and Mobile (Flutter) applications for the NeoWestMoney platform.

## Project Structure

- **[backend/](backend/)**: Go (Golang) backend service.
- **[frontend/](frontend/)**: Next.js web frontend.
- **[mobile/](mobile/)**: Flutter mobile application.
